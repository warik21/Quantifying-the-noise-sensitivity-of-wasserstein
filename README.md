# Quantifying the Noise Sensitivity of the Wasserstein Metric for Images

## Overview
This repository contains the source code and experimental scripts accompanying the paper **"Quantifying the noise sensitivity of the wasserstein metric for images."**

The code is provided here to facilitate the review process and allow for verification of the implementation details and results presented in the paper.

## Citation
